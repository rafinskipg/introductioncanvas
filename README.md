#DE CERO A CANVAS

![](https://github.com/rafinskipg/introductioncanvas/raw/master/cover.jpg)

Preparate para aprender de formar rápida y con ejemplos muchas de las posibilidades que puedes desarrollar usando Canvas.

En este libro podrás ver como: 
- Dibujar formas básicas, círculos, polígonos, líneas.
- Añadir sombreados, colores y gradientes a los elementos.
- Entender los principios básicos de la animación.
- Crear tu motor de renderizado en canvas.
- Crear animaciones de elementos.
- Crear diseños generativos. 
- Utilizar canvas en Node.
- Implementar partículas que interactuan entre sí con fuerzas (gravedad, repulsión, etc).
- Crear diferentes juegos básicos: **Juego de la Vida**, **Plataforma**
- Programar comportamientos emergentes.
